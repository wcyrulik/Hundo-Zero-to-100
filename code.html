The Code for THE HUNDRED



                        function getValues(){
                        
                            let startValue = document.getElementById("startValue").value;
                            let endValue = document.getElementById("endValue").value;
                            
                            // parse start and end values into integers
                            startValue = parseInt(startValue);
                            endValue = parseInt(endValue);
                        
                            if (Number.isInteger(startValue) && Number.isInteger(endValue)){
                                let numbers = generateNumbers(startValue,endValue);
                                displayNumbers(numbers);
                            } else {
                                alert("You must enter integers for start and end values");
                            }
                        }
                        
                        function generateNumbers(sValue, eValue) {
                            let numbers = [];
                        
                            for (let index = sValue; index <= eValue; index++) {
                                numbers.push(index);
                            }
                            return numbers;
                        }
                        
                        function displayNumbers(numbers){
                            let templateRows = "";
                        
                            for (let index = 0; index < numbers.length; index++) {
                                
                                let number = numbers[index];
                                let className = "even";
                        
                                if (number % 2 == 0){
                                    className = "even";
                                } else {
                                    className = "odd";
                                }
                        
                            // template literals do not render properly in prism, consequently,
                            // for a proper look at the following assignment of templateRows, 
                            // you will need to view site.js in the GitHub repo.
                            
                                templateRows += `${number}`;
                                
                            }
                            document.getElementById("results").innerHTML = templateRows;
                        }

The code is structured in three function.

getValues

This function is invoked by the click event listener defined in app.html. The functions receives input from the UI, validates that the inputs are integers, and calls the generateNumbers function, and the displayResults function.

generateNumbers

The generateNumbers function initializes an array called 'numbers' and then starts a loop from startValue to endValue. Inside the loop, the loop index is pushed (added) to the array numbers. When the loop terminates, the numbers array is returned to the calling function getValues.

displayResults

The displayResults function once again loops through the array of numbers. Each number is then evaluated to be either even or odd, and a css class name is assigned accordingly. A template literal is built up by concatenating a new table row to the growing string.

Once the loop completes the dom is updated by assigning the just constructed templateRow to the results element with the innerHTML function.
